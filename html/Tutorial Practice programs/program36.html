In order to set the width and height of element correctly in all browses , you need to know how the box model works.
Important : when you set the width and heigt properties of an elemet with CSS , you just set the width and height of the content area , to calculate the full sizeofan elemnet, you must also,add padding,borders and margins.

Calculate the total width : 320px(width) + 20px(left + right padding) + 10px(left + right margin) = 350px
Total elemaent width = width + left padding + right padding + left border + right border + left margin + right margin
Total element height = hight + left padding + right padding + left border + right border + left margin + right margin
